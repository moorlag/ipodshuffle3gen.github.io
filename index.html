Reverse Engineering the iPod Shuffle 3G Remote Protocol

Introduction

This page details the reverse engineering process of the iPod Shuffle 3rd generation's headphone remote protocol.

Disassembly

The first step was to disassemble the original remote to understand its construction and components.

Signal Investigation

Through experimentation, it was determined that specific resistor values correspond to different remote commands.

Resistor Values for Buttons

  • Play/Pause: 220 ohms
  • Next Track: 150 ohms
  • Previous Track: 360 ohms
  • Volume Up: 430 ohms
  • Volume Down: 820 ohms

Authentication Signal

Power-on authentication involves ultrasonic chirps.

Emulation

Successfully emulated using an ATmega88 microcontroller.

Future Development

The author plans to develop a low-power replacement for the original remote.
